This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Patient

Artifcats associated with Patients

BCPatient

General constraints on the Patient resource for use in the BC Client Registry project.

Operations

The defined Operations

FindCandidatesQuery

This operation is used to query for a patient. The response can contain 0 or more Paitents.
GetDemographicsQuery

This operation is used to query for a patient. The response can contain 0 or 1 Patient.
MergePatient

This operation is used to merge patients.
RevisePatient

This operation is used to revise a patient’s demographics.
RevisePatient.Newborn

This operation is used to create a newborn patient.

Operation Parameters

The parameter profiles associated with the Client Registry Operations

BCMetadataParameters

Parameters profile for BC meta data
BCPatientBusinessParameters

Parameters extended from BCMetadataParameters for a Patient resource

Response Bundles

A set of Bundle profiles used by the Client Registry to respond to searches and revise/merge requests

BCReviseAndMergeResponseBundle

A Bundle that is used in the Client Registry response to Revise and Merge Patient requests.
BCSearchResponseBundle

A Bundle that is used in the Client Registry response to Find Candidates and Get Demographics queries.
